## Die Size Overview

| Die Size | Typical Weight Range | Tablet Appearance | Swallowability | Best Powder Type | Common Uses |
|----------|---------------------|-------------------|----------------|------------------|-------------|
| **φ6mm** | 80–200mg | Small, easy to swallow | ★★★★★ | Dense minerals | Children's vitamins, melatonin, potent herbals |
| **φ8mm** | 180–400mg | Medium-small, comfortable | ★★★★ | Medium to dense | Single herbs, nootropics, caffeine |
| **φ10mm** | 350–700mg | Standard supplement size | ★★★ | Any density | Multi-herb blends, daily supplements |
| **φ12mm** | 600–1000mg | Large, substantial | ★★ | Light, bulky herbs | Meal supplements, fiber, greens powders |

## Tablet Weight Formula

```
Tablet Weight (mg) = π × (Diameter/2)² × Thickness × Powder Density
```

### Density Examples

| Powder Type | Approximate Density | φ10mm × 5mm Thickness |
|-------------|-------------------|----------------------|
| Dicalcium phosphate | ~1.2 g/cm³ | ~470mg |
| Herbal powder (turmeric) | ~0.8 g/cm³ | ~314mg |
| Spirulina | ~0.6 g/cm³ | ~235mg |

## Die Size Selection Flowchart

```
What's your active ingredient dose per tablet?
│
├── <150mg  → φ6mm (micro-dose)
├── 150–300mg → φ8mm (small dose)
├── 300–500mg → φ10mm (standard dose)
└── >500mg → φ12mm (large dose)

Adjust for powder density:
├── Light, bulky herbs (spirulina, maca) → Go UP one size
├── Dense minerals (calcium, magnesium) → Go DOWN one size
└── Targeting elderly/children → Go DOWN one size
```
